# Screenshots

Dieser Ordner enthält Screenshots für die README.md.

## Benötigte Screenshots

Nach dem Start der App, erstelle folgende Screenshots:

1. **screenshot-landing.png** - Landing Page mit Matrix-Hintergrund
2. **screenshot-dashboard.png** - Dashboard mit Kapiteln und Statistiken
3. **screenshot-exam.png** - Prüfungssimulation Startseite
4. **screenshot-flashcards.png** - Lernkarten mit SRS-Optionen

## Screenshot erstellen

### Windows
- `Win + Shift + S` - Ausschnitt speichern
- Oder: Snipping Tool verwenden

### Mac
- `Cmd + Shift + 4` - Ausschnitt speichern
- Oder: Screenshot App verwenden

### Linux
- `gnome-screenshot -a` - Ausschnitt speichern
- Oder: Flameshot verwenden

## Empfohlene Größe

- Breite: 1920px oder 1280px
- Format: PNG
- Dateiname: Kleinbuchstaben mit Bindestrich
